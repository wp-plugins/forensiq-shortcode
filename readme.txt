Creates a Forensiq shortcode to manage Forensiq Javscript. Find out <a href="http://forensiq.com/" target="_blank">more here</a>. 

== Installation ==

1. Unzip and Upload the folder 'cookie-params' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings and configure the options eg. Copy and paste the Google Adsense code that you want to use.
4. Add the shortcode to a post or page where you want it to appear.
